# Tilting Maze

# Description
- Users will control a ball through a maze with the use of a joystick.
- The goal is to combine the ball with the others to the center of the maze while avoiding obstalces.
- Users are also able to change the background.

# Features
- Ball movement
- Background changing buttons

- CSS
